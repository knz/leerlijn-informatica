Inloggen op een andere machine
------------------------------

Op de meeste machines gaat inloggen via SSH (:def:`secure shell`). Daarvoor heb je een SSH client nodig. Als je Linux of een Mac gebruikt, dan is een SSH client standaard al aanwezig: :program:`ssh`. Download voor Windows bijvoorbeeld :program:`PuTTY` (specifiek: :program:`putty.exe`).
